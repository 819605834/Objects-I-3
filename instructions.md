Write a function called `removeValue` which takes two parameters:

- obj - an object
- key - a string

The function should remove the given key from the object.

**For example:**

```
var obj = {
```

```
  'key': 'value',
```

```
}
```

```
removeValue(obj, 'key');
```

```
console.log(obj['key']); // should print undefined
```
